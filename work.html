Projects

DocuMinder - Desktop Document Expiry Date Reminder Application

DocuMinder is a desktop application for you to remind the important expiry dates of your documents in an organized manner. The application is purely Java written. It allows you to create/edit/delete the record of the document you have entered.

Imtor - A Drawing Robot

Imtor is a drawing robot made using the LEGO Mindstorms Kit. The robot is programmed using Python. A Scalable Vector Graphics (SVG) image is inputted and it's drawn on the paper by plotting the co-ordinates of the image. The robot can be used as a handcraft tool. The future development of this robot can bring a dramatic change in the mechanical writing efficiency of a robotic arm. (P.S :- It can write your school/college notes too!)

Guess Me - iOS Mobile Game

GuessMe is a simple number pattern based game. The game is intended for people who love numbers. It uses the latest programming language from Apple : Swift. The game contains 10 levels with beautiful UI of geometrical triangles.

Wasit Wetland Center Info App

Wasit Wetland Centre is a conservation area in Sharjah, United Arab Emirates. I made an iOS info app of Wasit Wetland which competed in the Sharjah Sustainability Award competition.
